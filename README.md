# Armon Nasiri Portfolio

Standalone React/Vite source for Armon Nasiri's public portfolio and Product Radar site.

Live site: https://armonnasiri.netlify.app/

## Local Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm run lint
npm run build
```

## Notes

- `public/downloads/` contains the public test packs and alpha artifacts linked from Product Radar.
- Netlify functions live in `netlify/functions/`.
- Generated folders such as `dist/`, `.netlify/`, and `node_modules/` are intentionally ignored.
